# Project Topic: 
최근 5년이내 서울시 자동차 현황 및 인사이트


## Project Description: 

최근 친환경 정책이 전 세계적으로 추진되고 있는데, 
서울시의 자동차 등록 현황을 보면 전체 자동자 등록수는 감소 추시지만
그 중 친환경 자동차의 보유량은 증가 추세로 확인되고 있는것으로 확인되어
전기차의 충전소는 잘 갖춰지고 있는지 앞으로 전기차는 얼마만큼 증가할 것인지 
예측하였고
이러한 분석으로 확인된 결과 
2022년 충전기 1대당 전기가 1.45씩 대응하던 것이 
2024년은 충전기 1대당 전기차 1.03대씩 대응하는것으로 
전기차의 이용이 수월해 졌고

현재 전체 자동차 중 전기차는 % 정도 차지하고 있는데,
향후 5년 후 o% 증가로 보여지고 있다.

앞으로도 친환경 자동차가 증가할 것으로 보인다
