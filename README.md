# Project Topic: 
최근 서울시 자동차 시장 트랜드 및 인사이트 도출


## Project Description:
최근 친환경 정책과 함께 국내 자동차 산업이 전통 내연기관을 넘어 미래차, 
특히 친환경 자동차 중심으로 빠르게 전환되는 시기로 전기차·하이브리드의 시장 확대, 
정부의 보조금 정책 변화, 글로벌 경쟁 환경 등이 맞물리면서 자동차 시장에 변화가 나타나고 있다.
서울시의 최근 3~4년간의 자동차 시장을 분석하여 향후 친환경 자동차의 변화를 분석하여
미래예측을 하는 프로젝트로 시작하였으며,
지금은 전기차로 넘어가는 과도기로 친환경 차종 중 하이브리드가 가장 많은 비중을 차지하고 있으며
서울시의 전기차의 비중도 매년 빠르게 증가하고 있다.
미래 예측 결과 향후 5년까지도 지속 증가하는것드로 보여졌다.
앞으로 자동차 시장은 정부의 정책과 더불어 
빠르게 친환경 자동차의 시장으로 확대되고 있음을 알 수 있다.
